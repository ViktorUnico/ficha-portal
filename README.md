# Ficha de Captação · Imobiliária Portal de Pirassununga

Ficha de captação de imóvel digital: preenche, mostra a prévia (frente e verso)
e gera PDF pra assinar + JPG. Funciona no computador e no celular.

- Os dados preenchidos ficam SÓ no navegador de quem usa (localStorage). Nada sobe pra internet.
- Arquivo único (`index.html`): pode ser usado online ou copiado pra qualquer máquina.

Ferramenta do Viktor Web Studio pro cliente Imobiliária Portal.
